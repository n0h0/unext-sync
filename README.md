# unext-sync

U-NEXTの動画**再生状態のみ**（play / pause / seek / playbackRate）を複数ユーザー間で同期するChrome拡張（MV3）とWSリレーサーバー。動画データ・映像・音声・DRMは一切共有・取得しない。各参加者は自分のアカウントで自分の配信ストリームを再生し、その再生位置をホストに追従させる「完全スレーブ方式」。

詳細な設計・実装方針は `CLAUDE.md` および `docs/superpowers/specs/` を参照。
